# Trading RL System

Reinforcement learning trading system built using:
- Gymnasium
- Stable-Baselines3 PPO
- yfinance
- Technical indicators
- KMeans pattern discovery

## Features

- OHLCV feature engineering
- Pattern discovery from historical data
- Custom RL trading environment
- R-multiple reward design
- PPO agent training
- Backtesting metrics and visualizations

## Metrics

- Total return
- Sharpe ratio
- Max drawdown
- Win rate
- Average R-multiple
